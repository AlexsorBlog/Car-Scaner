ELM327 Emulator & React App - Usage Guide

=== 1. ЗАПУСК ЕМУЛЯТОРА (Різні авто) ===
Емулятор підтримує різні "сценарії" (scenarios), які імітують різні типи транспортних засобів. 
Щоб запустити емулятор, відкрийте термінал у папці проєкту та використовуйте прапорець `-s` для вибору авто, а `-n 35000` для відкриття порту для нашого моста.

Варіанти запуску:
1. Toyota Auris Hybrid (Рекомендовано для тестів, найбільше підтримуваних команд):
> python -m elm -s car -n 35000

2. Мотоцикл / Квадроцикл (Delphi MT05 ECU):
> python -m elm -s mt05 -n 35000

3. Базовий набір команд (За замовчуванням):
> python -m elm -n 35000

=== 2. ПОРЯДОК ЗАПУСКУ ІНШИХ КОМПОНЕНТІВ ===
Крок 1: Запустіть міст TCP-до-WebSocket (у новому вікні терміналу)
> node tcp-bridge.js

Крок 2: Запустіть React додаток (у ще одному вікні)
> npm run dev
(Відкрийте http://localhost:5173/test у браузері)

=== 3. КОМАНДИ КОНСОЛІ ЕМУЛЯТОРА (CMD>) ===
Коли працює Python-емулятор, ви можете керувати ним прямо через його термінал.

Щоб змінити авто "на льоту" (без перезапуску):
CMD> scenario car
CMD> scenario mt05

Щоб протестувати команду (подивитися, що віддасть емулятор):
CMD> test 03
CMD> test 04

Щоб змінити налаштування відображення (якщо потрібно):
CMD> counters
CMD> loglevel debug

=== 4. ЯК СТВОРЮВАТИ ТА СТИРАТИ ПОМИЛКИ (CHECK ENGINE) ===
Щоб помилки можна було стерти через React-додаток (кнопкою "СТЕРТИ ПОМИЛКИ"), скопіюйте і вставте ці три команди в консоль емулятора CMD>:

# 1. Прив'язуємо команду читання (03) до динамічної змінної
emulator.answer['GET_DTC'] = '<exec>emulator.shared.dtc if hasattr(emulator.shared, "dtc") else "43 00"</exec><writeln />'

# 2. Прив'язуємо команду стирання (04) до очищення цієї змінної
emulator.answer['CLEAR_DTC'] = '<exec>setattr(emulator.shared, "dtc", "43 00")</exec><writeln>44</writeln>'

# 3. "Ламаємо" машину (записуємо дві помилки: P0104 та P0300)
setattr(emulator.shared, "dtc", "43 02 01 04 03 00")

Тепер у React-додатку:
1. Натисніть "ЗЧИТАТИ ПОМИЛКИ" -> З'явиться червоний список помилок.
2. Натисніть "СТЕРТИ ПОМИЛКИ" -> React відправить команду 04, змінна в Емуляторі очиститься.
3. Натисніть "ЗЧИТАТИ ПОМИЛКИ" -> Екран зелений, помилок немає!

*Примітка: Web Bluetooth працює лише в Chrome/Edge та вимагає HTTPS або localhost.*
